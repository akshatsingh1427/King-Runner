## Overview

**King Runner** is a focused **prototype** built to validate the core gameplay mechanics of the upcoming project Fun Bros.  
It emphasizes **movement, timing, and level flow** through a short, finite running level.

This project intentionally avoids feature complexity and instead concentrates on **strong fundamentals**.

---

## Gameplay

- Automatic forward movement
- Player-controlled jumping
- Obstacle avoidance
- Clear start and finish points
- Short **150-meter** handcrafted level

The game is **level-based**.

---

## Design Goals

- Validate core runner mechanics  
- Test jump responsiveness and collision flow  
- Evaluate short-level pacing  
- Serve as a technical and design foundation for **Fun Bros**

---

## Key Features

- Clean, minimal game loop  
- Smooth character movement  
- Simple but readable level layout  
- Lightweight and fast-loading web build  
- Prototype-focused architecture  

---

## Project Status

 **Prototype Stage**

This version represents an early gameplay proof-of-concept.  
Future updates may include enemies, collectibles, sound design, UI polish, and expanded levels.
